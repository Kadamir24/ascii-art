# ascii-art
Ascii-art consists on receiving a string as an argument and outputting the string in a graphic representation of ASCII.

# Usage
```
$ go build
$./ascii-art "hello"

  _              _   _
 | |            | | | |
 | |__     ___  | | | |   ___
 |  _ \   / _ \ | | | |  / _ \
 | | | | |  __/ | | | | | (_) |
 |_| |_|  \___| |_| |_|  \___/
```
